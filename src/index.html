Qpang is back! Project AnimeShooter is a unofficial server for QPang (aka MangaFighter), the game is a multiplayer online third-person shooter developed by OnNet and hosted by game portal GamesCampus. Players will join other combatants in a 3D manga-style world, with multiple modes of combat and a variety of character customization options. The game first began its open beta on July 31, 2007, and was released to the public on December 3, 2007. The game was shut down on July 30, 2010. A Dutch version of the game remained in operation until June 2013 under the name QPang.

AnimeShooter, the longest running Qpang private server, aims to re-create the original Qpang experience from 2013 by providing servers emulators that follow the exact protocol of the original game. This allows you to use the original Qpang game files (along with our patcher) to connect to our server emulator and play Qpang like it was 2013!

Hopefully, you are just as excited as we are to enjoy Qpang once again after roughly 8 years since the original game was discontinued! Unlike the original developers, we plan ahead and aim for many more years of Qpang to come!

Register

To get started, you must create an account at our website. Once created, you will be guided with instructions to download and install the game and how you can connect to our game servers.

Please keep in mind that we are providing our very own game servers that come with their very own database. This means all progress from the original game is NOT there. To obtain in-game items you can use in-game currencies or real-life currencies to purchase in-game goods.

How to Connect● GameServer

  1. Download and run QPangSetup.exe
  2. Take note where Qpang gets installed
  3. Download QPangBin.zip and extract in Qpang folder (default path: C:\Program Files (x86)\RealFogs\QPang\)
  4. Done, now launch Qpang with run QPangBin_Launcher.bat

Patch Files

The original Qpang binary still houses HShield (anti-cheat) and thus must be modified to get the game working. To do so, QPangBin.zip is provided with already modified game files.

For those who have trust issues and prefer to patch the files themselfs, you can do so using the QPangMiniLauncher GitHub Repo. The list of patches to be applied can be found in QPangMiniLauncher's Program.cs
